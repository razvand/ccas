Vector Functions of Several Variables​

A vector-valued function of several variables is a function

f:Rm→Rnf: \mathbb{R}^{m} \rightarrow \mathbb{R}^{n}f:Rm→Rn

Suppose the real valued function f:Rm→Rf:\mathbb{R}^m \rightarrow \mathbb{R}f:Rm→R is differentiable in each coordinate. Then the gradient of fff, denoted ∇f\nabla f∇f is given by
