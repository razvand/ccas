Numbers, Arithmetic and Basic Algebra

Natural Numbers​

The positive integers are called natural numbers. These numbers can be added, multiplied together and so forth.

Notation: N={1,2,3,4,… }\mathbb{N}=\{1,2,3,4,\dots \}N={1,2,3,4,…}

Subtraction and division are not defined on these numbers.

An arbitrary element of N\mathbb{N}N is most commonly denoted by i,j,ni, j, ni,j,n, or mmm, but any symbol can be used.

Details​

Definition

The set of positive integers is usually denoted by N\mathbb{N}N, i.e. N={1,2,3,4,… }\mathbb{N}=\{1,2,3,4,\dots \}N={1,2,3,4,…} and is called the set of natural numbers. In some cases the number zero is included as a natural number, but here we will use the symbol N0\mathbb{N}_0N0​ to denote the integers 0, 1, 2 and up.

Within this set of numbers it is possible to add and multiply numbers together. Arithmetic operations are denoted by +++ for addition and ⋅\cdot⋅ (or ×\times×) for multiplication. A natural number can also be raised to the power of a natural number, e.g. 35=3⋅3⋅3⋅3⋅33^5=3\cdot 3\cdot 3\cdot 3\cdot 335=3⋅3⋅3⋅3⋅3 or in general mn=m⋅m⋅…⋅mm^n=m\cdot m \cdot \ldots \cdot mmn=m⋅m⋅…⋅m ( nnn times).

When stating general properties of the natural numbers one needs to use symbols to indicate that the property holds for an arbitrary number. It is not enough to just write the property for a few numbers. For example, to declare that one can interchange numbers in a sum, it is not enough to say 4+3=3+44+3=3+44+3=3+4, but one must explicitly state "the addition operator has the property that any two natural numbers, n, m∈Nn,\ m\in \mathbb{N}n, m∈N satisfy n+m=m+nn+m=m+nn+m=m+n".

An arbitrary element of N\mathbb{N}N is most commonly denoted by i, j, ni,\ j,\ ni, j, n,or mmm, but any symbol, a, b, c,…a,\ b,\ c, \ldotsa, b, c,…, can be used.

Several rules of arithmetic apply (some by definition, others can be derived) such as

ab=baa+b=b+aa+bc=a+(bc)a(b+c)=ab+ac(a+b)+c=a+(b+c)(ab)c=a(bc)\begin{aligned} ab &= ba \\ a+b &= b+a \\ a+bc &= a+(bc) \\ a(b+c) &= ab+ac \\ (a+b)+c &= a+(b+c) \\ (ab)c &= a(bc) \end{aligned}aba+ba+bca(b+c)(a+b)+c(ab)c​=ba=b+a=a+(bc)=ab+ac=a+(b+c)=a(bc)​

Subtraction and division are not generally defined. In addition, we define one integer, nnn, to the power of another, mmm, to mean nnn multiplied by itself mmm times: nm=n⋅n⋅…⋅n⏟mn^m=\underbrace{n\cdot n \cdot \ldots \cdot n}^{m}nm=n⋅n⋅…⋅n​m.

Definition

The power is an operator just like addition and multiplication, and is defined to have higher priority than the other two.

Examples​

Example

If we have x=4x=4x=4 and y=2y= 2y=2 and want to evaluate:

xy+yxx^y+y^xxy+yx

then we replace the values of x and y in the expression, and evaluate it, taking care to observe the correct order of operations:

42+24=16+16=324^2+2^4=16+16=3242+24=16+16=32

The Integers​

The set of positive and negative integers:

Z:={…,−2,−1,0,1,2,… }\mathbb{Z} := \{\dots, -2, -1, 0, 1, 2, \dots \}Z:={…,−2,−1,0,1,2,…}

Details​

Definition

The set of all integers is denoted by Z\mathbb{Z}Z, i.e.

Z:={…,−2,−1,0,1,2,… }\mathbb{Z} := \{\dots, -2, -1, 0, 1, 2, \dots \}Z:={…,−2,−1,0,1,2,…}

Note

Note that within this set it is possible to subtract as well as add and multiply. Within this set we cannot, however, in general, perform division.

When preforming multiple mathematical operations within the same equation, i.e. 79−8⋅379 - 8\cdot 379−8⋅3, there is a conventional order for which the operations must be performed.

Definition

The conventional order of operations for equations with multiple mathematical operations is referred to as an operator precedence.

Examples​

Example

To compute 79−8⋅379 - 8\cdot 379−8⋅3 start by multiplying and then subtracting:

79−8⋅3=79−24=5579 - 8\cdot 3 = 79-24 = 5579−8⋅3=79−24=55

Example

To compute

15−(24+36)15 - (24 + 36)15−(24+36)

we first note that the parentheses (brackets) imply a precedence; anything inside brackets should be evaluated first. Thus, we first add 363636 to 242424 and then we subtract that from 151515:

15−(24+36)=15−60=−4515 - (24+36) = 15 - 60 = - 4515−(24+36)=15−60=−45

Note that the answer is a negative number.

Example

Simple arithmetic in R is easily done at the command prompt:

> 79-8*3
[1] 55

> 15-(24+36)
[1] -45

Rational Numbers​

Rational numbers are fractions denoted pq\displaystyle\frac{p}{q}qp​, where ppp and qqq are integers. We can simplify fractions if the numerator and denominator contain common terms.

Details​

Definition

Rational numbers are fractions denoted p/qp/qp/q, where ppp and qqq are integers. The set of all rational numbers is usually denoted Q\mathbb{Q}Q.

Note

Note that every integer is a rational number (obtained by taking q=1q=1q=1).

We can simplify fractions if the numerator and denominator contain common terms. When the rationals are ordered on to a line there are points missing, i.e. there are "gaps"; for example there is no rational number p/qp/qp/q such that (p/q)2=2(p/q)^2=2(p/q)2=2.

Examples​

Example

26=22⋅3=13\displaystyle\frac{2}{6} = \displaystyle\frac{2}{2 \cdot 3} = \displaystyle\frac{1}{3}62​=2⋅32​=31​

The rational numbers can be put in order along a line as in the figure.

Example

As an elaborate example of a fraction, consider the evaluation of the quantity:

23+2513+12\displaystyle\frac{\displaystyle\frac{2}{3}+\displaystyle\frac{2}{5}}{\displaystyle\frac{1}{3}+\displaystyle\frac{1}{2}}31​+21​32​+52​​

Example

Evaluate:

23+2513+12\displaystyle\frac{\displaystyle\frac{2}{3}+\displaystyle\frac{2}{5}}{\displaystyle\frac{1}{3}+\displaystyle\frac{1}{2}}31​+21​32​+52​​

Solution: We can either start by calculating the numerator:

23+25\displaystyle\frac{2}{3}+\displaystyle\frac{2}{5}32​+52​

or the denominator:

13+12\displaystyle\frac{1}{3}+\displaystyle\frac{1}{2}31​+21​

Here we choose to start with the numerator. The first step is to make the two fractions in the numerator have a common denominator.

We can either find the least common denominator or use the product of the two denominators. Here they are the same number, 15.

So the first step is:

23⋅55+25⋅33=displaystyle2⋅53⋅5+2⋅35⋅3=1015+615\displaystyle\frac{2}{3} \cdot \displaystyle\frac{5}{5} + \displaystyle\frac{2}{5} \cdot \displaystyle\frac{3}{3} = \\displaystyle\frac{2 \cdot 5}{3 \cdot 5} + \displaystyle\frac{2 \cdot 3}{5 \cdot 3} = \displaystyle\frac{10}{15} + \displaystyle\frac{6}{15}32​⋅55​+52​⋅33​=displaystyle3⋅52⋅5​+5⋅32⋅3​=1510​+156​

Now it is possible to add the two fractions, which is the second step:

10+615=1615\displaystyle\frac{10+6}{15} = \displaystyle\frac{16}{15}1510+6​=1516​

Next, the same process has to be performed for the original denominator.

With the same method (LCM - least common multiple) we get:

1⋅23⋅2+1⋅32⋅3=26+36=56\displaystyle\frac{1 \cdot 2}{3 \cdot 2} + \displaystyle\frac{1 \cdot 3}{2 \cdot 3} = \displaystyle\frac{2}{6} + \displaystyle\frac{3}{6} = \displaystyle\frac{5}{6}3⋅21⋅2​+2⋅31⋅3​=62​+63​=65​

Then the total answer is:

161556=1615⋅65=9675=96/375/3=3225\displaystyle\frac{\displaystyle\frac{16}{15}}{\displaystyle\frac{5}{6}} = \displaystyle\frac{16}{15} \cdot \displaystyle\frac{6}{5} = \displaystyle\frac{96}{75} = \displaystyle\frac{96/3}{75/3} = \displaystyle\frac{32}{25}65​1516​​=1516​⋅56​=7596​=75/396/3​=2532​

We can see that in the last step of the equation, the factor has been simplified. To do this we use factoring. Here we obtain:

9675=3⋅323⋅25\displaystyle\frac{96}{75} = \displaystyle\frac{3 \cdot 32}{3 \cdot 25}7596​=3⋅253⋅32​

We can now remove 333, or the multiplier, as it is on both sides of the fraction. So we have:

3225=2525+725=3225\displaystyle\frac{32}{25} = \displaystyle\frac{25}{25} + \displaystyle\frac{7}{25} = \displaystyle\frac{32}{25}2532​=2525​+257​=2532​

In step 1 above we used Cross-Multiplication.

Definition: Cross-Multiplication is when we multiple the numerator by the reciprocal of the denominator.

So in this case we rewrite

161556\displaystyle\frac{\displaystyle\frac{16}{15}}{\displaystyle\frac{5}{6}}65​1516​​

or

1615÷56\displaystyle\frac{16}{15} \div \displaystyle\frac{5}{6}1516​÷65​

as

1615⋅65\displaystyle\frac{16}{15} \cdot \displaystyle\frac{6}{5}1516​⋅56​

As you can see all we are doing is turning

56\displaystyle\frac{5}{6}65​

upside down; and multiplying it with

1615\displaystyle\frac{16}{15}1516​

This gives

9675\displaystyle\frac{96}{75}7596​

In some cases it is possible to draw a square root of a fraction s=pqs = \displaystyle\frac{p}{q}s=qp​, i.e. find a number r∈Qr\in \mathbb{Q}r∈Q such that r2=sr^2=sr2=s. The square root is denoted r\sqrt{r}r​.

Example

Consider the expression

(19⋅24)+(15⋅25)\left(\sqrt{\displaystyle\frac{1}{9}} \cdot 2^4\right) + (\displaystyle\frac{1}{5} \cdot \sqrt{25})(91​​⋅24)+(51​⋅25​)

To evaluate this expression, first consider separately the two parts on each side of the plus symbol.

The first part is

(19⋅24)\left(\sqrt{\displaystyle\frac{1}{9}} \cdot 2^4\right)(91​​⋅24)

and the second part is

(15⋅25)(\displaystyle\displaystyle\frac{1}{5} \cdot \sqrt{25})(51​⋅25​)

In addition, by definition of root,

19=13\sqrt{\displaystyle\frac{1}{9}} = \displaystyle\frac{1}{3}91​​=31​

First part:

(19⋅24)=13⋅16=163\left(\sqrt{\displaystyle\frac{1}{9}} \cdot 2^4\right) = \displaystyle\frac{1}{3} \cdot 16 = \displaystyle\frac{16}{3}(91​​⋅24)=31​⋅16=316​

Second part:

(15⋅25)=15⋅5=1(\displaystyle\frac{1}{5} \cdot \sqrt{25}) = \displaystyle\frac{1}{5} \cdot 5 = 1(51​⋅25​)=51​⋅5=1

Finally, add the first part and the second part:

163+1=193\displaystyle\frac{16}{3} + 1 = \displaystyle\frac{19}{3}316​+1=319​

Example

Consider the following fraction example, to be solved step by step:

42+(14l⋅53)26÷15\displaystyle\frac{\displaystyle\frac{4}{2} + (\displaystyle\frac{1}{4l} \cdot \displaystyle\frac{5}{3})}{\displaystyle\frac{2}{6} \div \displaystyle\frac{1}{5}}62​÷51​24​+(4l1​⋅35​)​

First we need to be aware of operator precedence, sometimes called BODMAS (brackets, multiplication/division, then addition/subtraction).

(14⋅53)=512(\displaystyle\frac{1}{4} \cdot \displaystyle\frac{5}{3}) = \displaystyle\frac{5}{12}(41​⋅35​)=125​

After solving the bracket we can proceed with adding

42\displaystyle\frac{4}{2}24​

to

512\displaystyle\frac{5}{12}125​

as there is no other action left for the nominator of the main fraction.

So:

42+512\displaystyle\frac{4}{2} + \displaystyle\frac{5}{12}24​+125​

When adding fractions together we first have to find a common denominator, in this case 12 would work as

2⋅6=122\cdot6=122⋅6=12

So we multiply both the numerator and the denominator of that fraction by 6 and then add the two numerators of the fractions together, keeping the same denominator.

42+512=4⋅62⋅6+512=2412+512=2912\displaystyle\frac{4}{2} + \displaystyle\frac{5}{12} = \displaystyle\frac{4 \cdot 6}{2 \cdot 6} + \displaystyle\frac{5}{12} = \displaystyle\frac{24}{12} + \displaystyle\frac{5}{12} = \displaystyle\frac{29}{12}24​+125​=2⋅64⋅6​+125​=1224​+125​=1229​

Now we have the top half of the fraction solved. We then proceed with dividing the two fractions of the bottom half. When dividing fractions we use the so called cross multiplication technique. This arithmetic trick is derived from the fact that if you divide a fraction by its duplicate you get 1. If you multiple a fraction by its reciprocal (it's reverse) you also get 1. Like so:

12÷12=1\displaystyle\frac{1}{2} \div \displaystyle\frac{1}{2}=121​÷21​=1

and

12⋅21=1\displaystyle\frac{1}{2} \cdot \displaystyle\frac{2}{1}=121​⋅12​=1

These functions always provide the same result and therefore we can turn the fraction we are dividing by upside down and multiply it to the other fraction as that is usually much easier.

We can therefore rewrite

26÷15\displaystyle\frac{2}{6} \div \displaystyle\frac{1}{5}62​÷51​

as

26⋅51=106\displaystyle\frac{2}{6} \cdot \displaystyle\frac{5}{1} = \displaystyle\frac{10}{6}62​⋅15​=610​

We've now solved both halves of the original fraction and can therefore proceed to solve it, again with the cross multiplication technique as fractions are after all just divisions:

2912÷106=2912⋅610=174120\displaystyle\frac{29}{12} \div \displaystyle\frac{10}{6} = \displaystyle\frac{29}{12} \cdot \displaystyle\frac{6}{10} = \displaystyle\frac{174}{120}1229​÷610​=1229​⋅106​=120174​

Now

174120\displaystyle\frac{174}{120}120174​

is a pretty bad looking fraction and we'd preferably like to simplify it. To do this we use factoring.

Definition: Factoring essentially means to break a number done into its smallest factors or multipliable prime numbers.

In this case we get

2⋅3⋅292⋅3⋅20\displaystyle\frac{2 \cdot 3 \cdot 29}{2 \cdot 3 \cdot 20}2⋅3⋅202⋅3⋅29​

These are the smallest prime numbers that can multiply together into 174 and 120 respectively.

A way of doing this in your head is by first dividing both numbers (174 and 120) by 2. Which gives us:

2⋅872⋅60\displaystyle\frac{2 \cdot 87}{2 \cdot 60}2⋅602⋅87​

and then dividing those numbers (87 and 60) by 3, since they can't be divided by 2. Dividing by 3 gives you

3⋅293⋅20=2920\displaystyle\frac{3 \cdot 29}{3 \cdot 20} = \displaystyle\frac{29}{20}3⋅203⋅29​=2029​

which is a lot nicer than

174120\displaystyle\frac{174}{120}120174​

The reasoning behind this factoring simplification is that we can remove multipliers if they are on both sides of a fraction. This is because the result of a fraction where the numerator and the denominator are the same is always 1. Like so:

11=1\displaystyle\frac{1}{1}=111​=1

or

22=1\displaystyle\frac{2}{2}=122​=1

or

33=1\displaystyle\frac{3}{3}=133​=1

The final answer therefore is

The Real Line​

Some obvious numbers are not fractions. The set of numbers making up the real line is denoted by the symbol R\mathbb{R}R.

Figure: The diagonal of a rectangle with unit side lengths of 2\sqrt{2}2​. Note that 2\sqrt{2}2​ is not a fraction.

Details​

Some obvious numbers, which commonly occur, are not fractions. These are in between the rational numbers (fractions) on the number line. Filling in the missing points to obtain a continuum results in the set of real numbers.

Denoted by R\mathbb{R}R the entire set of real numbers which corresponds to filling in the missing pieces of the number line, so to speak.

Examples​

Example

If CCC is the circumference of a circle and DDD is the diameter and we define π=CD\pi = \displaystyle\frac{C}{D}π=DC​ then π\piπ is not a fraction.

Example

One example of a non-fraction is the number e (Euler's number) which can be defined by:

e=∑n=0∞1n!e = \sum_{n=0}^{\infty }\displaystyle\frac{1}{n!}e=∑n=0∞​n!1​

Example

If you have a right triangle with unit side length, what is the length of its hypotenuse and what class of numbers does it belong to?

An isosceles triangle is defined as having adjacent and opposite sides of same length, connected by a 90∘90^{\circ}90∘ angle. Unit side length of these, refers to a side length of 1.

As we have a 90∘90^{\circ}90∘ angle, we can use Pythagoras' theorem:

a2+b2=c2a^2+b^2=c^2a2+b2=c2

With

a=adjacenta=\text{adjacent}a=adjacent

b=oppositeb=\text{opposite}b=opposite

c=hypotenusec=\text{hypotenuse}c=hypotenuse

So with

a,b=1:a,b=1:a,b=1:

c2=12+12c^2=1^2+1^2c2=12+12

c2=1+1c^2=1+1c2=1+1

c2=2c^2=2c2=2

We take the square root to get ccc:

c=2c=\sqrt{2}c=2​

Now that we answered the first part of the question, it needs to be defined, which class of number 2\sqrt{2}2​ belongs to. 2\sqrt{2}2​ is an irrational number, and belongs thereby to the set of real numbers R\mathbb{R}R. Real numbers can be imagined as points on an infinitely long line, which is also called the real line.
