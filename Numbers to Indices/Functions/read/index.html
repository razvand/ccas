Functions

Functions of a Single Variable​

A function describes the relationship between variables.

Examples:

f(x)=x2f(x) = x^2f(x)=x2

y=2+3⋅x4y = 2+3\cdot x^4y=2+3⋅x4

Details​

Functions are commonly used in statistical applications, to describe relationships.

Definition

A function describes the relationship between variables. A variable yyy is described as a function of a variable xxx by completely specifying how yyy can be computed for any given value of xxx.

An example could be the relationship between a dose level and the response to the dose.

The relationship is commonly expressed by writing either f(x)=x2f(x) = x^{2}f(x)=x2 or y=x2y = x^2y=x2.

Usually names are given to functions, i.e. to the relationship itself. For example, fff might be the function and f(x)f(x)f(x) could be its value for a given number xxx. Typically f(x)f(x)f(x) is a number but fff is the function, but the sloppy phrase "the function f(x)=2x+4f(x)=2x+4f(x)=2x+4" is also common.

Examples​

Example

f(x)=x2f(x) = x^2f(x)=x2 or y=x2y = x^2y=x2 specifies that the computed value of yyy should always be x2x^2x2, for any given value of xxx.

Ranges and Plots in R​

Functions in R can commonly accept a range of values and will return a corresponding vector with the outcome.

Examples​

Example
f <- function(x) {return(x*12)}
x <- seq (-5,5,0,1)
y <- f(x)
plot {(x,y) type= 'l'}

Plotting Functions​

In statistics, the function of interest is commonly called the response function. If we write y=f(x)y=f(x)y=f(x), the outcome yyy is usually called the response variable and xxx is the explanatory variable. Function values are plotted on vertical axis while xxx values are plotted on horizontal axis. This plots yyy against xxx.

Examples​

Example

The following R commands can be used to generate a plot for function y=2+3xy= 2+3xy=2+3x:

x <- seq(0:10)
g <- function(x) {
+ yhat <- 2+3*x
+ return(yhat)
+ }

x <- seq(0,10,0.1)
y <- g(x)
plot(x,y,type="l", xlab="x",ylab="y")
